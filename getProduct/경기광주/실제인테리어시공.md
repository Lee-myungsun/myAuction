## 1. 철거 사전 준비
* 철거 대상 확인 -> 작업대상임
   * ```  거실바닥, 각 방, 욕실 2개, 현관중문, 주방싱크대, 몰딩, 각 방문 등   ```
   * 철거에 필요한 장비 확인

* 셀프 철거 진행
   * ```  화장실, 주방, 기타 등등   ```
* 철거 업체 도움
   * 아파트 바로 엽 철거 업체: https://blog.naver.com/pis8767  
   * 바닥 철거 업체 활용 
   
* 욕실 철거
  * 타일 덧방, 바닥 방수 체크
  * 욕실 신발 높이(70mm이상)

## 2. 설비 작업 확인
#### 각종 수전 체크
* 젠다이 연장배관, 메꾸라, 서비스 니플
 * 양수놈 스덴 12cm  15파이프
 * 15mm 신주소켓
 * PVC(물) 35mm 배관 및 엘보 
 *  ``` 수도배관 사이 길이 체크, 세면더 뒷변 높이가 작거나, 폭이 큰거 선택 ```
* 세탁실 및 건조기 배관
* 주방 배관 및 수전 체크
* 난방 설비 체크
  * 분배기 노후도 체크  

#### 체크 사항
* 욕실, 배란다 배수 트렌치는 기본 사각형
* 환풍기가 제대로 되는지 확인(천장)
* 환풍기와 조명은 분리(전기에서 확인)
* 천장은 SMC 평면 천장
* 젠다이 설치시 연장할때 누수체크
* 바닥 타일은 미끄럽지 않은 타일
